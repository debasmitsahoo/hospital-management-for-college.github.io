# Hospital Management System (HMS)

A modern, responsive, and interactive Hospital Management System web application built entirely with **Vanilla JavaScript, HTML, and CSS**. No external frameworks or backends are used.

## ğŸš€ Features

*   **Modern UI/UX**: Clean, responsive interface with a medical color scheme, smooth transitions, and a professional landing page.
*   **Role-Based Authentication**: Simulated login for Admin, Doctors, and Staff with varying access levels.
*   **Single Page Application (SPA) Dashboard**: Smooth navigation without page reloads using dynamic Javascript rendering.
*   **Patient Management**: Add, View, and Delete patient records.
*   **Doctor Management**: Manage doctor profiles and specializations.
*   **Medical Records**: Create and view medical history linked to patients and doctors.
*   **Data Persistence**: All data is stored locally in your browser (LocalStorage), so it survives page refreshes.
